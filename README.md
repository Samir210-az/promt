# Promt.AI 🎭

Zarafatcıl AI Prompt Generatoru — sadə fikrini yaz, AI əvvəlcə zarafat eləsin, sonra sənə hərtərəfli, peşəkar AI promtu hazırlasın.

## Xüsusiyyətlər
- 🤖 Groq API (llama-3.3-70b-versatile) ilə işləyir — pulsuz
- 😄 Hər sorğudan sonra mövzuya uyğun zarafat
- 📝 Ümumi AI, şəkil, kod, video, yazı üçün fərqli promt formatları
- 📱 PWA — telefona quraşdırıla bilər, offline interfeys işləyir
- 🎨 Parallax, animasiyalar, rəngli keçidlər

## İstifadə
1. Saytı aç
2. ⚙️ "AI ayarları" bölməsindən pulsuz Groq API açarını əlavə et ([console.groq.com/keys](https://console.groq.com/keys))
3. Fikrini yaz, "Promtu Yarat" düyməsinə bas

API açarı yalnız sənin brauzerində saxlanılır, heç bir serverə getmir.
